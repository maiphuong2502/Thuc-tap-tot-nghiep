I. User Flow Tổng Quan – Quản Lý bộ đề (Admin)

Admin truy cập Trang quản trị → Admin đăng nhập tài khoản quản trị → Hệ thống kiểm tra thông tin đăng nhập
Sai thông tin
→ Hiển thị lỗi đăng nhập
Đúng thông tin
→ Chuyển đến Dashboard quản trị
→ Admin chọn Quản lý bộ đề thi
→ Hệ thống truy vấn dữ liệu từ bảng tests và test_categories
→ Hiển thị Danh sách bộ đề
Thông tin hiển thị gồm:
• ID bộ đề
• Tên bộ đề
• Loại đề thi
• Thời gian làm bài
• Tổng số câu hỏi
• Trạng thái
• Người tạo
• Thời gian tạo
→ Admin có thể thực hiện các thao tác:
• Xem chi tiết
• Thêm bộ đề
• Chỉnh sửa bộ đề
• Xóa bộ đề
Hệ thống cập nhật dữ liệu
